What is the Document Object Model (DOM)?

the Document Object Model (Dom) is programming interface of the web document. When Web page loaded the browser create Document Object Model this page. The html Dom model is look like a tree of object. it defines some kind of how to access, modify and manipulate element using script language like JavaScript.

How do you select an element from the DOM?

DOM can be selected in many ways, some of them are methods:
1. select By ID: document.getElementById('id');
2. select By Class Name: document.getElementByClassName('ClassName');
3. select By Tag Name: document.getElementsByTagName("tagName");

CSS Selectors in querySelector and querySelectorAll:

4. select By Css Selector(First Match): document.querySelector("cssSelector");
5. select By Css Selector(All Match): document.querySelectorAll("cssSelector");
6. select By Class name (single class): document.querySelectorAll(".class");
7. select By id: document.querySelectorAll(".class");
8. select By tag: document.querySelectorAll("tagName");
9. select By child elements: document.querySelectorAll("parent > child");
10. select By sibling elements: document.querySelectorAll("element1 + element2");
